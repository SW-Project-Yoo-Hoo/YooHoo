## <img src="https://github.com/SW-Project-Yoo-Hoo/YooHoo/assets/71424881/d4aa1857-b74a-4d66-ab9a-c02ac46eb54f" width="40px"> 프로젝트 소개 
__YooHoo__ 는 유휴자원으로 남아 있는 사무 용품을 공유하는 웹 플랫폼입니다.
자원을 방치하는 것이 아닌 필요한 사람에게 공유함으로써 자원을 순환시킵니다.
유휴자원을 소유한 기업은 부가 수익을 창출하고, 대여하는 기업은 경제적 부담을 덜 수 있습니다.
공유의 방식으로 소유가 가지는 문제점을 개선하고 __자원의 지속적인 순환__ 을 목표로 합니다.

#### [YooHoo 바로가기 :rocket:](http://43.201.179.41:3000)

## 💡 주요 기능

<h3>물품 등록하기</h3>

- 게시물 사진/정보/카테고리/대여 기간/가격 등의 정보를 담은 게시물을 등록합니다.

<h3>거래하기</h3>

- 수요자 : 게시물 상세 보기 페이지에서 대여 기간 선택 후 거래를 요청합니다.
- 공급자 : 받은 요청 페이지에서 원하는 요청을 수락합니다.

<h3>반납하기</h3>

- 조기 반납 : 대여 기간이 만료되지 않았을 때도 조기 반납 버튼을 통해 반납할 수 있습니다.
- 반납 : 대여 기간이 만료되었을 때 반납 버튼을 통해 반납합니다.
- 반납 완료 : 공급자와 수요자가 모두 반납 확인 버튼을 눌렀을 때 반납이 완료되고 거래가 종료됩니다.

<h3>정보 조회하기</h3>

- 알람 : 요청/반납/대여 기간 등의 알람이 제공됩니다.
- 회원 정보 : 프로필 페이지에서 확인 가능하며, 수정할 수 있습니다.
- 게시물 정보 : 프로필 페이지에서 내 게시물/요청 현황/반납 현황/받은 요청/거래 현황 등의 게시물 관련 모든 정보를 확인할 수 있습니다.

<h3>추천받기</h3>

- 수요자 : 게시물 전체보기에서 특정 기간/카테고리에서 가장 저렴한 게시물을 추천받습니다.
- 공급자 : 받은 요청에서 특정 기간 내에 수익이 가장 큰 게시물의 조합을 추천받습니다.
